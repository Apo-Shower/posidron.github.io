Tripbit Security Advisory
TA-270503 


Severity: 
Medium/High
Application: 
 Abyss Web Server X1 Update 1 v1.1.5
Platform: 
 Windows
Class: 
 Directory Traversal Vulnerability
Release Date: 
 May 27th, 2003
Vendor: 
http://aprelium.com


• Overview
Abyss is a very good free and tiny webserver that not only has a lot of functions but it also run on both Windows and Linux systems.

• Details
Possible bug or misconfiguration problem in the web server that allow unauthorized remote users to gain information about the web server's host machine, that will allow them to break into the system.

Example:

http://host.com/cgi-bin/?_DIR_OUT=.|.%2F.|.%2F.|.%2F.|.%2F.|.%2F.|.%2F.|.%2F.|.%2F.|.%2F.|

• Recommendation
No solution for the moment.

• Vendor Response
The vendor has reportedly been notified but no answer to this report.
