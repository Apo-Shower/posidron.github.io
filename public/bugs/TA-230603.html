Tripbit Security Advisory
TA-230603 


Severity: 
Medium/High
Application: 
 iWeb Server 2
Platform: 
 Windows
Class: 
 Directory Transversal Vulnerability
Release Date: 
 June 20th, 2003
Vendor: 
http://www.ashleybrown.co.uk/iweb/


• Overview
The iWeb Mini Web Server is a mini web server designed for use on Intranets and for testing websites in a realistic environment.

• Details
There is an other Directory Transversal Vulnerability in iWeb Server which allows an remote attackers to see the content of the requested file.

Example:

http://host/%5c%2e%2e%5c%2e%2e%5c%2e%2e%windows\system.ini

• Recommendation
No solution for the moment.

• Vendor Response
The vendor has reportedly been notified.
