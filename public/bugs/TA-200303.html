Tripbit Security Advisory
TA-200303 


Severity: 
Medium/High
Application: 
 Hyperion FTP Server 2.7 <= 3.0
Platform: 
 Windows
Class: 
 Insufficient Boundary Checking
Release Date: 
 March 20th, 2003
Vendor: 
http://www.mollensoft.com


• Overview
Hyperion FTP Server is a small FTP Server Program for Windows. It is made by Mollensoft and it supports all FTP Server commands.

• Details
A security vulnerability in Hyperion FTP server allows remote attackers to cause the server to crash by executing a specific command (mkdir) with a buffer of 256 Bytes and more in length. The command can be issued to the FTP server either by a valid authenticated user or by an guest/anonymous account (if this is enabled).

• Recommendation
No solution for the moment.

• Vendor Response
The vendor has reportedly been notified, but no answer to this report.
